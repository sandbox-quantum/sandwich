# bazelisk

[Documentation](https://docs.rs/bazelisk)

A build dependency for running [Bazel] using the [Bazelisk] binaries to compile
native libraries and more.

```toml
# Cargo.toml
[build-dependencies]
bazelisk = "0.1.0"
```
